# 📊 Customer Visualization using t-SNE

## 📌 Overview

This project uses t-SNE to reduce high-dimensional customer data into 2D for visualization and pattern discovery.

---

## 🎯 Objective

Understand customer behavior by visualizing complex data in a simplified form.

---

## 🧠 Model

t-SNE (t-Distributed Stochastic Neighbor Embedding)

---

## ⚙️ Technologies

* Python
* Pandas
* Scikit-learn
* Matplotlib

---

## 🚀 How to Run

pip install -r requirements.txt
python main.py

---

## 📈 Output

* 2D visualization of customers
* Cluster-based grouping

---

## 💡 Business Impact

* Identify customer segments
* Improve marketing strategies
* Discover hidden patterns

---

## 🎯 Learning Outcomes

* Dimensionality reduction
* Data visualization
* Preprocessing techniques
